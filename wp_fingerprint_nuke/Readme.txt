== Description ==
This plugin Does some various modifications to wordpress on the frontend to reduce the fingerprint that you are using wordpress, It does this without you modifying your theme files. The things it does include: 
 * REMOVE wordpress emoji script
 * REMOVE RSD link header 
 * REMOVE feed links
 * REMOVE comments feed
 * REMOVE meta generator
 * REMOVE wordpress version appended in scripts
 * Disables wordpress REST API 
 * Disable wordpress user enumeration
 * Remove unnecessary Junk in header
